02 skills&tools

html

  • 웹 표준에 맞춰 문서구조를 작성할 수 있음
  • 웹 접근성에 맞춰 문서를 작성할 수 있음
  • semectic 태그를 사용하여 문서구조를 작성할 수 있음
  • 그리드시스템에 맞춰 문서구조를 작성할 수 있음

css

  • 선택자를 명확히 선택할 수 있음
  • 가상요소를 적재 적시에 활용할 수 있음
  • key frame을 통한 애니메이션 구현가능
  • 반응형 웹 구현가능(크로스 브라우져, 크로스 디바이스)
  • 적응형 웹 구현가능(크로스 브라우져, 크로스 디바이스)

scss

  • 선택자를 정확히 선택할 수 있음
  • 가상요소를 적재 적시에 활용할 수 있음
  • 변수, 반복문, 조건문 활용을 통한 선택자 및 스타일 속성 간소화 가능
  • mixin활용을 통한 스타일 속성 간소화 가능
  • minimize 작업을 통한 파일 용량 줄이기 가능

javascript

  • DOM탐색을 명확히 할 수 있음
  • 일반슬라이드, 무한루프 슬라이드 레퍼런스 없이 구현가능
  • fullpage기능 레퍼런스 없이 구현가능
  • ES6/ES5사용가능
  • 동기, 비동기 작업 처리 조절 가능

jquery

  • DOM탐색을 명확히 할 수 있음
  • 일반슬라이드, 무한루프 슬라이드 레퍼런스 없이 구현가능
  • fullpage기능 레퍼런스 없이 구현가능
  • ES6/ES5사용가능
  • 동기, 비동기 작업 처리 조절 가능
  • jquery cookie를 활용할 수 있음
  • ajax통신을 지정할 수 있음
  • load를 활용한 컴포넌트의 모듈화 가능

hive/SQL

  • hadoop에서 데이터 조회 및 엑셀로 피벗가공가능
  • oracle에서 데이터 조회 및 엑셀로 피벗가공가능
